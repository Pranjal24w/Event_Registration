#Event Registration System (Flask + MySQL)

A simple mini-project where students register for events and admins can view total registrations.

Features
Student registration
View events
Store data in MySQL
Admin dashboard (participants per event)
Very easy to run & submit

📁 Folder Structure
event_registration/
│── app.py
│── templates/
│   ├── index.html
│   ├── admin.html
│── static/
│   └── style.css
│── requirements.txt

Prerequisites
Python 3
MySQL Server
Install dependencies:
pip install flask mysql-connector-python
python app.py

Open in browser:

Student Page → http://127.0.0.1:5000
Admin Page → http://127.0.0.1:5000/admin
